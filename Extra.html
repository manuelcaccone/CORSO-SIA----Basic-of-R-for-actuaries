    • GitHub Copilot

  • Vantaggi: produttività, riduzione errori, accessibilità

GitHub Copilot

Cos'è GitHub Copilot?

  • Assistente AI sviluppato da GitHub e OpenAI
  • Basato su modelli linguistici avanzati (GPT-4)
  • Funzionalità:
    • Suggerimento di codice
    • Completamento funzioni
    • Documentazione automatica
    • Generazione script

Integrazione con RStudio

  • Nativa in RStudio Desktop (versione 2023.09.0+)
  • Disponibile su Posit Workbench/RStudio Server

Requisiti

  • Account GitHub con abbonamento Copilot attivo
    • Gratuito per studenti/docenti
  • RStudio Desktop aggiornato
  • Connessione internet

Installazione e Configurazione

  1. Apri RStudio → Tools → Global Options → Copilot
  2. Abilita "Enable GitHub Copilot"
  3. Scarica componenti aggiuntivi
  4. Accedi con account GitHub

Per RStudio Server/Workbench: aggiungi copilot-enabled=1 in /etc/rstudio/rsession.conf

Funzionalità

  • Completamento automatico di codice e funzioni
  • Suggerimenti contestuali
  • Supporto multilingua: R, Python, SQL, HTML, CSS, JavaScript
  • Documentazione automatica
  • Adattamento allo stile di programmazione

Esempio pratico

# Calcola la media di una colonna in un dataframe

Copilot suggerirà automaticamente il codice R corrispondente.

Vantaggi e Limiti

Vantaggi:

  • Riduce errori di sintassi e logica
  • Aumenta la produttività
  • Utile per tutti i livelli di esperienza

Limiti:

  • Richiede abbonamento
  • Necessita connessione internet
  • Non sempre perfetto su codice personalizzato

groqR

Cos'è groqR?

  • Pacchetto R per integrare modelli AI di Groq
  • Supporta DeepSeek-R1, Llama-3, Mixtral e altri
  • Noto per velocità e ottimizzazione hardware

Caratteristiche chiave

  • Inferenza ultra-veloce (fino a 10x rispetto a GPU tradizionali)
  • Supporto multi-modello
  • Privacy di livello enterprise
  • Piano gratuito disponibile
  • Add-in per RStudio

Installazione

  1. Crea account su Groq
  2. Ottieni API Key dalla dashboard
  3. In R:
install.packages("groqR")
library(groqR)
groq_set_api_key("TUACHIAVEAPI")
  1. Avvia l'add-in di groqR in RStudio

Funzionalità

  • Chat AI per domande su codice, debug
  • Traduzione codice tra linguaggi
  • Generazione automatica di test e documentazione
  • Suggerimenti per nomi di variabili/funzioni
  • Analisi e sintesi di dati

Esempio pratico

groq_chat("Scrivi una funzione R che normalizza una colonna di un dataframe tra 0 e 1.")

Vantaggi e Limiti

Vantaggi:

  • Risposte quasi istantanee
  • Adatto a flussi di lavoro intensivi
  • Buona privacy

Limiti:

  • Richiede API Key e account esterno
  • Modelli specifici di Groq

gander

Cos'è gander?

  • Add-in open source sviluppato da Simon Couch (Posit)
  • Chat AI direttamente in RStudio
  • Caratteristica distintiva: accede al contesto dell'ambiente R

Funzionalità principali

  • Chat AI contestuale con informazioni su oggetti, dati e codice attivi
  • Integrazione con modelli OpenAI, Anthropic, Ollama (tramite ellmer)
  • Risposte personalizzate basate su stile e contenuto del progetto
  • Add-in richiamabile da tastiera

Installazione

# Installa ellmer (backend per modelli AI)
install.packages("ellmer")

# Installa gander
install.packages("gander")

Configura API Key per OpenAI, Anthropic o Ollama
Avvia da RStudio (Addins → gander)

Come funziona

  1. Seleziona codice o posizionati dove vuoi assistenza
  2. Attiva l'add-in e scrivi la richiesta
  3. gander invia automaticamente il contesto al modello AI
  4. Ricevi risposta dettagliata e contestuale

Esempio pratico

Con un dataframe df nell'ambiente, chiedi:

"Mostrami un esempio di visualizzazione dei dati di df"

gander capirà la struttura e proporrà un esempio coerente.

Vantaggi e Limiti

Vantaggi:

  • Risposte pertinenti e personalizzate
  • Nessun copia/incolla di codice o dati
  • Perfetto per analisi, debugging, tutoring

Limiti:

  • Ancora in fase iniziale (alcuni bug)
  • Richiede configurazione API Key
